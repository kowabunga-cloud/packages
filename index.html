To install Kowabunga packages, add a new repository to /etc/apt/sources.list.d/kowabunga.repo.list:

deb http://packages.kowabunga.cloud/ ubuntu main

And import key that is used to sign the release:

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 5277E8C721237125

After that you can install Dalet packages as any other software package:

sudo apt-get update
sudo apt-get install PACKAGE
